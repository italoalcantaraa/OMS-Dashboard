# 📦 OMS Dashboard

O **Order Management System (OMS) Dashboard** é uma interface que entrega uma visão geral de todos os pedidos de compra de uma loja (dados mockados do [mockapi.io](https://mockapi.io)), trazendo informações rápidas como:

- ✅ Pedidos entregues
- 🚚 Pedidos em trânsito
- ❌ Pedidos cancelados

Além disso, permite:

- 🔍 Visualização detalhada de cada pedido de compra
- 👤 Filtro por nome do cliente
- 📋 Filtro por status do pedido

---

## 🎯 Propósito do Projeto

Compreender os conceitos fundamentais de **ReactJS com TypeScript**, aplicando boas práticas de desenvolvimento para criar códigos mais seguros, organizados e legíveis.

---

## 🛠️ Tecnologias Utilizadas

- ReactJS
- Vite
- TypeScript
- MockAPI
